## What is this?

Formula 1 defines success by one metric: the World Drivers' Championship. But what about the drivers who never won — not because they lacked talent, but because they never had the car?

This project analyses **non-champions from 2005–2025** using three car-neutral metrics to separate driver skill from machinery. The analysis was inspired by Hülkenberg's maiden podium at the 2025 British GP — his first in 239 career starts.

## Key Findings

| Rank | Driver | Composite Score | Why |
|------|--------|----------------|-----|
| 🥇 | Mark Webber | 0.867 | Highest teammate delta in the dataset (+3.57). 42 podiums. 2010 title lost in Abu Dhabi. |
| 2nd | Leclerc | 0.779 | Active driver — career far from over |
| 3rd | Ricciardo | 0.596 | 2014: outqualified and outscored Vettel in the same car |
| 4th | Kubica | 0.593 | Career cut short by 2011 rally accident at his peak |
| 19th | Hülkenberg | 0.469 | The inspiration. 239 starts, 1 podium. Some stories are bigger than their numbers. |

## Metrics

| Metric | Definition | What it captures |
|--------|-----------|-----------------|
| **Podium Rate** | Podiums ÷ completed races | Peak performance ceiling |
| **Overperformer Delta** | Driver avg finish vs constructor avg finish | Beating the car |
| **Teammate Delta** | Avg qualifying gap vs same-car teammate | Pure talent, car-neutral |

All three metrics normalised (Min-Max, 0–1) and averaged into a composite score. Minimum threshold: 20 race finishes.

## How to Run

```bash
# Clone the repo
git clone git@github.com:costa911/f1-analysis.git
cd f1-analysis

# Set up environment
python3 -m venv venv
source venv/bin/activate
pip install jupyter requests pandas matplotlib seaborn scikit-learn

# Launch notebook (data fetches live from Jolpica API — allow 10–15 mins)
jupyter notebook f1_analysis.ipynb

# Regenerate charts
python charts.py
```

> ⚠️ The API fetches ~21 seasons of data across 5 endpoints. Expect 10–15 minutes for a full run due to rate limiting (`time.sleep()` included).
